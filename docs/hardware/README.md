# Hardware Reference

This directory keeps hardware diagnostics for `portajohn`, the ASUS Eee PC 1005HA managed by this repo.

Use the sanitized report before adding packages or services. The point is to keep provisioning grounded in the machine's actual limits: i386 only, about 1 GB RAM, old Intel 945GSE graphics, low-resolution panel, and very limited CPU headroom.

The report is sanitized before committing to remove machine identifiers such as serial numbers, UUIDs, MAC addresses, and disk serials while preserving capability and performance details.
